Orc Engine
=
##介绍
Orc Engine是一个基于[redis](http://redis.io/)的"轻量级、高扩展性、快速"推荐引擎.
通过Orc你可以快速创建基于用户行为的内容推荐系统.

##安装条件
```
nodejs >= 4.0.0
redis >= 2.0.0
```

##测试
```
npm test
```
